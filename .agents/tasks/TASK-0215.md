Status: DONE

## Objective
Expose deterministic owner analytics derived from event-store facts (no LLM dependency).

## Endpoint
- `GET /owner/insights/summary`
- Query:
  - `tenant_id` (required)
  - `since_ts` (optional)
  - `until_ts` (optional)
- Auth:
  - `Authorization: Bearer <VOZ_OWNER_API_KEY>`

## Feature gate
- `VOZ_FEATURE_OWNER_INSIGHTS=0|1`

## Behavior
- Tenant-scoped counts:
  - `flow_a.call_started`
  - `flow_a.call_stopped`
  - `flow_a.transcript_completed`
  - `postcall.summary`
  - `postcall.lead` total
  - `postcall.lead` where `qualified=true`
  - `postcall.appt_request`
- Window defaults:
  - if missing, default to last 24h
  - bounded to max 7 days
- Includes latest `(rid, ts)` within window.
- Debug-only signature:
  - `OWNER_INSIGHTS_SUMMARY tenant_id=... since_ts=... until_ts=...`

## Acceptance coverage
- Correct counts from seeded events
- Missing/invalid bearer -> `401`
- Tenant isolation enforced
